Grace Framework

Hi, and thanks for checking out Grace Framework! Grace is a hybrid Java/JavaScript web framework aiming to pair the power of the JVM and the rich ecosystem of Java libraries with the dynamic goodness of the universal language of the web, JavaScript. Grace Framework is modeled after its tiny namesake, who is:

Small

Grace is purposefully small and optimized for building MVC web applications and JSON web services on GAE.

Beautiful

API design is important. Grace's APIs strive to be both terse and intuitive, allowing you to write beautiful code.

Intelligent

Grace tries to do the right things by default, but gives you raw request/response level visibility into handling a servlet request when you need it.

Helpful

We know you're busy, so Grace tries to help you along with the pieces of functionality you'll need to build MVC web applications and RESTful JSON services. Grace provides the kind of dynamic, RESTful routing you'd expect from a modern web framework, plus a JavaScript-based template engine to keep your view code DRY. Not that you need to use any of it - the smallest Grace app can be 3 lines of code.

Acknowledgements

Grace draws inspiration from great web frameworks like Ruby on Rails and Sinatra. Grace uses the Mozilla Rhino JavaScript engine, and leverages the reference JSON JavaScript library, as well as the very snazzy Underscore utility library.
